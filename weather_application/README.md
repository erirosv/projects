# A simple live update weather app

```
src/
├── weather_fetcher.py       # Main script to fetch & store weather data
├── config.py                # API key and config
├── database.py              # DB setup
├── scheduler.py             # Scheduled job runner
├── weather_log.csv          # Optional: stores data in CSV
├── weather_data.db          # SQLite database
└── streamlit_app.py         # Optional: Live dashboard
```

```bash 
pip install -r requirements.txt
```
